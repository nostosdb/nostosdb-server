# Contributor license agreement status

This repository does not offer or accept a contributor license agreement. The
project does not accept external contributions, so no public CLA or DCO process
is operated.

Copyright holder Jongho Yoo is the only receiving owner. A patch, pull request,
DCO sign-off, issue comment, or other unsolicited submission is not accepted as
a contribution grant and must not be merged or copied as one. This file is a
policy statement, not an agreement or invitation to submit work.
